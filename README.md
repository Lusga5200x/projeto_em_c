## Como o jogo funciona

O jogo consiste na comparação de duas cartas que representam cidades/estados, cada uma com diferentes atributos numéricos.

O jogador seleciona um atributo para comparação, como população, área, PIB ou densidade populacional. O programa então compara os valores entre as duas cartas e determina a vencedora com base no maior ou menor valor, dependendo do atributo escolhido.

## Fluxo do jogo:
- As cartas são previamente cadastradas no código
- O jogador escolhe um atributo de comparação
- O sistema exibe os valores das duas cartas
- A comparação é feita automaticamente
- O programa informa a carta vencedora
- Regra de vitória:
    - Vence a carta com o maior valor
    - Exceto a densidade populacional, que vence o menor valor

## ✅ Funcionalidades já implementadas

- Cadastro de duas cartas (cidades)
- Entrada de dados via teclado
- Armazenamento de:
  - Estado
  - Código da cidade
  - Nome
  - População
  - Área
  - PIB
  - Pontos turísticos

- Cálculos automáticos:
  - Densidade populacional
  - PIB per capita
  - Super poder da cidade

- Exibição completa das cartas
